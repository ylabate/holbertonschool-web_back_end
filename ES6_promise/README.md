# ES6 promise

## 0-promise.js
Creating a basic Promise that resolves immediately using Promise.resolve().

## 1-promise.js
Creating a Promise constructor that resolves or rejects based on a success parameter.

## 2-then.js
Using then() and catch() methods to handle Promise results and errors with appropriate logging.

## 3-all.js
Using Promise.all() to handle multiple Promises and wait for all to resolve before processing results.

## 4-user-promise.js
Creating a Promise that resolves with user information (firstName and lastName).

## 5-photo-reject.js
Creating a Promise that rejects with an error message when a photo cannot be processed.

## 6-final-user.js
Using Promise.allSettled() to handle multiple Promises regardless of their fulfillment state and map results.

## 7-load_balancer.js
Using Promise.race() to return the result of whichever Promise resolves first (load balancing).

## 8-try.js
Creating a function that divides numbers with error throwing for invalid operations.

## 9-try.js
Implementing a guardrail pattern using try-catch to handle function execution safely and return status messages.
