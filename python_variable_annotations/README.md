# python variable annotations

## 0-add.py
Function that adds two floats with type annotations and returns their sum.

## 1-concat.py
Function that concatenates two strings with type annotations and returns the result.

## 2-floor.py
Function that takes a float and returns its rounded integer value with type annotations.

## 3-to_str.py
Function that converts a float to a string with type annotations.

## 4-define_variables.py
Script that defines and annotates variables of different types: integer, float, boolean, and string.

## 5-sum_list.py
Function that takes a list of floats and returns their sum with proper type annotations using typing.List.

## 6-sum_mixed_list.py
Function that takes a list of mixed integers and floats and returns their sum as a float.

## 7-to_kv.py
Function that takes a string and a numeric value, returning a tuple with the string key and the squared value.

## 8-make_multiplier.py
Higher-order function that returns a multiplier function with proper type annotations using typing.Callable.

## 9-element_length.py
Function that takes an iterable of sequences and returns a list of tuples containing each sequence and its length.
