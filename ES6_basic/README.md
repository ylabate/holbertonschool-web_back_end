# ES6 Basic

## 0-constants.js
Basic constant and let variable declarations for understanding block-scoped variables.

## 1-block-scoped.js
Exercise in preventing variable overwriting using block scope (let and const).

## 2-arrow.js
Converting standard function syntax to ES6 arrow functions with lexical this scoping.

## 3-default-parameters.js
Implementing default values for function parameters to handle missing arguments.

## 4-rest-parameter.js
Using the rest parameter syntax to handle an indefinite number of function arguments.

## 5-spread-operator.js
Using the spread operator to concatenate arrays and copy object properties.

## 6-string-interpolation.js
Implementing template literals for cleaner string concatenation and variable interpolation.

## 7-getBudgetObject.js
Utilizing object property value shorthand to simplify object creation.

## 8-getBudgetCurrentYear.js
Implementing computed property names to define object keys dynamically.

## 9-getFullBudget.js
Using ES6 method properties to define functions within objects concisely.

## 10-loops.js
Iterating through arrays using the ES6 for...of loop syntax.

## 11-createEmployeesObject.js
Creating department-specific employee objects using computed property names and spread syntax.

## 12-createReportObject.js
Generating a comprehensive report object with employee lists and department counting methods.
