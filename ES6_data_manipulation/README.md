# ES6 data manipulation

## 0-get_list_students.js
Creating a function that returns a static array of student objects with id, firstName, and location properties.

## 1-get_list_student_ids.js
Extracting student IDs from an array of student objects using the map method with error handling.

## 2-get_students_by_loc.js
Filtering students by location using the filter method to find all students in a specific city.

## 3-get_ids_sum.js
Calculating the sum of all student IDs using the reduce method for array aggregation.

## 4-update_grade_by_city.js
Updating student grades by city, filtering students and mapping new grades with fallback handling for missing grades.

## 5-typed_arrays.js
Working with TypedArrays and DataView to create an Int8 typed array with specific value assignment at a position.

## 6-set.js
Converting an array to a Set data structure to remove duplicates and work with unique values.

## 7-has_array_values.js
Checking if all elements from an array exist in a Set using the every method for validation.

## 8-clean_set.js
Processing a Set to extract and filter values based on a string prefix, returning formatted output.

## 9-groceries_list.js
Creating and returning a Map data structure containing grocery items with their quantities.

## 10-update_uniq_items.js
Updating Map items that have a value of 1 to 100, with type checking and error handling for non-Map inputs.
